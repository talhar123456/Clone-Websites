Status

The freeBSD/ppc port is still a Tier 2 platform. That means it is not fully supported by our security officer, release engineers are toolchain maintainers.

Latest News

  • 03 April, 2009: Support for the PowerPC GS has been commited to SVN
  • 03 March, 2008: Support for Freescale® PowerQUICC MPC85XX family system-on-chip development boards has been commited into CVS
  • 27 February, 2008: FreeBSD 7.0 is the first release to officialy support the FreeBSD/ppc port
